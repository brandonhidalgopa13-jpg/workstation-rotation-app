COMANDOS PARA EJECUTAR DESPUÉS DE CREAR EL REPOSITORIO EN GITHUB:

1. Conectar con tu repositorio (reemplaza TU_USUARIO con tu nombre de usuario de GitHub):
   git remote add origin https://github.com/TU_USUARIO/workstation-rotation-app.git

2. Cambiar a rama main:
   git branch -M main

3. Subir el código:
   git push -u origin main

EJEMPLO:
Si tu usuario es "juan123", el comando sería:
git remote add origin https://github.com/juan123/workstation-rotation-app.git
